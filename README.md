# Проект: Путешествие по России

Учебный проект Яндекс Практикума по специальности "Веб-разработчик".

---

В проекте используются технологии и подходы:
1. CSS Flex и Grid разметка блоков
2. БЭМ (Nested)
3. Медиа запросы
4. Подход "Mobile First"
